Clasificador de Flores - Machine Learning

📖 Explicación del Caso de Uso

Este es un modelo de Machine Learning supervisado que clasifica flores en tres tipos: Setosa, Versicolor y Virginica. Se basa en características como el tamaño de los sépalos y pétalos. El modelo fue entrenado con el dataset Iris usando un algoritmo de Random Forest.

📝 ¿Qué valores puedo ingresar?

  • Longitud del sépalo: entre 4.3 y 7.9
  • Ancho del sépalo: entre 2.0 y 4.4
  • Longitud del pétalo: entre 1.0 y 6.9
  • Ancho del pétalo: entre 0.1 y 2.5
